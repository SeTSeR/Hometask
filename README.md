# Hometask [![Build Status](https://travis-ci.org/SeTSeR/Hometask.svg?branch=master)](https://travis-ci.org/SeTSeR/Hometask.svg?branch=master)

To create PDF files run:
```
make pdf
```

To create tex files run:
```
make tex
```
